CRC Model · Class Responsibility Collaborator

LUMS Campus Event Discovery App

Student Entity
Responsibilities
  • Knows own profile: name, student ID, department, year
  • Knows list of saved/bookmarked events
  • Knows RSVP'd events and attendance history
  • Knows preferred event categories and interests
  • Browse and search the event feed
  • RSVP to / cancel registration for an event
  • Save and unsave events for later
  • Receive and dismiss notifications
  • Follow societies and organizers
Collaborators
  • Event
  • EventFeed
  • Notification
  • Society
  • Tag
Event Entity
Responsibilities
  • Knows title, description, date, time, and duration
  • Knows hosting society or organizer
  • Knows assigned venue and capacity
  • Knows current RSVP count and attendee list
  • Knows associated tags and category
  • Knows approval status (pending / approved / rejected)
  • Check whether capacity is available
  • Accept or remove an RSVP
  • Display full event details to a viewer
  • Trigger notification on update or cancellation
Collaborators
  • Student
  • Society
  • Venue
  • Tag
  • Notification
  • Admin
Society Entity
Responsibilities
  • Knows society name, description, and category
  • Knows list of members and office bearers
  • Knows all events it has created (past & upcoming)
  • Knows follower count and list
  • Create a new event and submit for approval
  • Edit or cancel an existing event
  • Post announcements to followers
  • Manage its public profile and banner
Collaborators
  • Event
  • Venue
  • Student
  • Admin
  • Notification
  • Tag
EventFeed Service
Responsibilities
  • Knows the full collection of approved, upcoming events
  • Knows current filter state (date, category, society)
  • Knows active search query
  • Retrieve and display paginated list of events
  • Filter events by date range, tag, or society
  • Search events by keyword in title or description
  • Sort events by date, popularity, or relevance
  • Generate personalised recommendations for a student
  • Surface "trending" and "happening soon" events
Collaborators
  • Event
  • Student
  • Tag
  • Society
Venue Entity
Responsibilities
  • Knows name, building, room number, and map location
  • Knows maximum seating / standing capacity
  • Knows list of bookings by date and time slot
  • Knows available facilities (projector, mic, etc.)
  • Check availability for a requested date and time
  • Reserve a time slot for an event
  • Release a reservation when event is cancelled
  • Report conflicts to the requesting society
Collaborators
  • Event
  • Society
  • Admin
Notification Service
Responsibilities
  • Knows recipient student, message content, and type
  • Knows timestamp and read/unread status
  • Knows the event or society that triggered it
  • Send push notification to a student's device
  • Send event reminder 24 hrs before start time
  • Alert student when a followed society posts an event
  • Notify student when RSVP is confirmed or waitlisted
  • Mark notification as read when viewed
  • Batch and aggregate notification digests
Collaborators
  • Student
  • Event
  • Society
Tag Entity
Responsibilities
  • Knows tag name and category (e.g. Music, Tech, Sports)
  • Knows the set of events associated with this tag
  • Knows usage frequency across all events
  • Label an event with one or more tags
  • Provide a filterable list of all distinct tags
  • Support EventFeed in category-based filtering
  • Power interest-matching for student recommendations
Collaborators
  • Event
  • EventFeed
  • Student
  • Society
Admin Actor / Role
Responsibilities
  • Knows list of pending event approval requests
  • Knows registered societies and their status
  • Knows all venues and their booking schedules
  • Approve or reject a society's event submission
  • Feature or pin an event on the home feed
  • Register, suspend, or remove a society account
  • Manage and add new venues to the system
  • View platform-wide analytics and reports
  • Handle reported / flagged content
Collaborators
  • Event
  • Society
  • Venue
  • Notification
