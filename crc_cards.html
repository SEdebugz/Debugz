CRC Cards

Student Entity
Responsibilities
  • Knows name, student ID, department, year
  • Knows list of bookmarked events
  • Knows preferred categories
  • Knows own registrations and notification preferences
  • Browse and search the event feed
  • Register for or cancel registration for an event
  • View event details and schedule
  • Add approved events to a personal calendar
  • Receive and manage event notifications

  • EventFeed
  • Registration
  • CalendarIntegration
  • NotificationService
  • Organizer
  • Tag
Event Entity
Responsibilities
  • Knows title, description, date, time, and duration
  • Knows organizer and assigned venue
  • Knows current registration count, ticket availability
  • Knows associated tags, category, and status
  • Knows approval state of attendees if applicable
  • Check whether capacity is available
  • Open or close registration
  • Display full event details to a viewer
  • Track attendance and check-in totals
  • Update its status when changed or cancelled
  • Request notifications when details change

  • Venue
  • Tag
  • Registration
  • CalendarIntegration
  • NotificationService
  • Admin
Organizer Role
Responsibilities
  • Knows organizer profile, contact details, and permissions
  • Knows the events it has created and their statuses
  • Use the event management dashboard
  • Create a new event and submit it for approval
  • Edit, reschedule, or cancel an existing event
  • Publish updates to feed
  • Monitor registrations, attendance, and capacity usage

EventFeed Service
Responsibilities
  • Knows the full collection of approved, upcoming events
  • Knows current state and search query
  • Knows current sort mode and recommendations
  • Retrieve and display list of events
  • Filter events by required criteria
  • Search events by keyword in title or description
  • Sort events by date, popularity, or relevance
  • Generate personalised recommendations
  • Show trending and happening soon events
Collaborators
  • Event
  • Student
  • Tag
  • Organizer
Venue Entity
Responsibilities
  • Knows name, building, and room
  • Knows maximum seating
  • Knows list of bookings by date and time slot
  • Knows available facilities (projector, mic)
  • Check availability for a requested date and time
  • Reserve a time slot for an event
  • Release a reservation when event is cancelled
  • Report conflicts to the requester
Collaborators
  • Event
  • Organizer
  • Admin
Registration Entity
Responsibilities
  • Knows the student and event linked to a registration
  • Knows registration status, timestamp, ticket type, and waitlist position
  • Knows whether attendance was checked in
  • Create a registration for an eligible student
  • Cancel or restore a registration
  • Confirm whether a seat is reserved or waitlisted
  • Record check-in and attendance status
  • Provide registration details to Student and Event
Collaborators
  • Student
  • Event
  • NotificationService
CalendarIntegration Service
Responsibilities
  • Knows supported calendar providers
  • Knows which event details are required for calendar export
  • Add an event to a student's personal calendar
  • Update or remove calendar entries when event details change
  • Handle calendar API requests and sync failures
  • Prevent duplicate calendar entries for the same event

NotificationService Service
Responsibilities
  • Knows notification templates, channels, and delivery rules
  • Knows which event changes should trigger alerts
  • Create notifications for registrations, reminders, and cancellations
  • Send real-time updates to interested students
  • Mark notifications delivered or failed

Tag Entity
Responsibilities
  • Knows tag name and category
  • Knows which events are associated with the tag
  • Knows tag usage frequency
  • Label an event with one or more tags
  • Provide a list of available tags
  • Support EventFeed in category filtering
  • Support interests for recommendations

Admin Role
Responsibilities
  • Knows list of pending event approval requests
  • Knows registered organizers and their status
  • Knows all venues and their booking schedules
  • Approve or reject an organizer's event submission
  • Feature or pin an event on the home feed
  • Register, suspend, or remove an organizer account
  • Manage venue records and resolve scheduling conflicts
  • Handle reported or flagged content
